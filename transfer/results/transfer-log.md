# Transfer Experiment Log

All transfer experiments and their outcomes are recorded here.

## Log Format

Each entry follows this structure:

```

**Experiment file:** experiments/[filename].md
**Source agent:** [Agent]
**Target agent:** [Agent]
**Phase:** Before baseline / After run / Complete

### Scores

| Task | Dimension | Before | After | Delta |
|------|-----------|--------|-------|-------|
| ...  | ...       | ...    | ...   | ...   |

### Summary
[One paragraph: what happened, what was observed, what to do next]

### Status
[Not started / Transfer applied / Reverted / Complete]
```
